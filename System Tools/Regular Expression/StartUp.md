# 正则表达式

```txt
^ 为匹配输入字符串的开始位置。
[0-9]+匹配多个数字， [0-9] 匹配单个数字，+ 匹配一个或者多个
* 表示0个或多个
? 表示0个或者一个
$ 为匹配输入字符串的结束位置。
() 标记一个子表达式的开始和结束位置。子表达式可以获取供以后使用。要匹配这些字符，请使用 \( 和 \)。
{n} n 是一个非负整数。匹配确定的 n 次。
{n,} n 是一个非负整数。至少匹配n 次
{n,m} m 和 n 均为非负整数，其中n <= m。最少匹配 n 次且最多匹配 m 次。
```

## 将连续相同的字符转化为一个

```js
var reg = /(<p><br><\/p>)(\1)+/g;
var str = '<p><br></p><p><br></p>';
alert(str.replace(reg,'$1'));
```
